# Detecting Data Leaks Using SQL Injection

## CodeAlpha Cloud Computing Internship - Task 4

## Overview

Detecting Data Leaks Using SQL Injection is a cloud-based security solution developed as part of the CodeAlpha Cloud Computing Internship. The project is designed to protect web applications and cloud-hosted databases from SQL injection attacks while ensuring the confidentiality, integrity, and availability of sensitive user information.

The system integrates secure coding techniques, AES-256 encryption, capability-based access control, and a double-layer security mechanism to prevent unauthorized database access and minimize the risk of data leakage. It demonstrates how modern cloud applications can be designed with security as a primary objective.

## Objectives

- Prevent SQL Injection attacks using secure query execution.
- Protect sensitive user information through AES-256 encryption.
- Implement capability-based access control for secure server authorization.

## Installation

Clone the repository

```bash
git clone https://github.com/preethinalibiddi03-prog/Detecting-Data-Leaks-Using-SQL-Injection.git
```

Navigate to the project

```bash
cd Detecting-Data-Leaks-Using-SQL-Injection
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
